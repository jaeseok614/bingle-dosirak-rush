# 빙글도시락 캐논 제출 PDF 원고

## 1. 게임 개요

**빙글도시락 캐논**

쏘고, 키우고, 위 칸에 배달하는 60초 도시락 러시

빙글도시락 캐논은 수박게임식 합체를 쌓기 대신 조준 발사로 바꾼 60초 웹 미니게임입니다. 플레이어는 지금 재료를 쏘고, 중앙의 같은 재료를 맞춰 키운 뒤, 완성된 음식을 위쪽 같은 재료 칸에 배달합니다.

- 플레이 링크: https://jaeseok614.github.io/bingle-dosirak-rush/
- 실행 방식: 브라우저에서 바로 실행되는 정적 웹앱
- 핵심 규칙: 쏘기 -> 키우기 -> 배달

## 2. 문제의식과 기획 의도

흔한 웹 미니게임은 점프, 피하기, 클릭 반복에 머무는 경우가 많습니다. 이 게임은 짧은 시간 안에 이해되는 60초 타임어택 안에 합체, 조준 발사, 주문 배달을 결합했습니다.

- 수박게임식 합체를 "쌓기"가 아니라 "조준해서 성장시키기"로 재해석했습니다.
- 복잡한 보관함이나 회수 조작 없이 지금 재료 하나에 집중하도록 단순화했습니다.
- 첫 10초 안에 "중앙 재료를 맞춰 키우고 위 칸에 배달한다"는 규칙이 보이도록 설계했습니다.

## 3. 핵심 플레이 루프

1. 지금 재료를 발사합니다.
2. 중앙의 같은 재료를 맞춥니다.
3. 지금 재료가 한 단계 성장합니다.
4. 완성되면 위쪽 같은 재료 칸에 배달합니다.
5. 연속 성공으로 콤보와 피버 자동발사를 노립니다.

초반에는 밥 배달과 주먹밥 만들기부터 시작해 튜토리얼 없이 바로 시작해도 규칙을 익힐 수 있습니다.

## 4. 재미 요소

- 직배송: 정확하게 위 칸에 바로 넣으면 추가 보상
- 빙글배송: 벽을 튕겨 넣는 물리 기반 하이라이트
- 도시락 도장: 배달 성공 순간의 즉각적인 피드백
- 콤보: 합체와 배달을 끊기지 않고 이어갈수록 점수 상승
- 피버 자동발사: 조건 달성 시 잠깐 강한 자동발사
- 화살표 부스터: 중반 이후 등장해 재료 경로를 바꾸는 변수
- 마감 러시: 마지막 10초에 긴장감을 높이는 연출

## 5. UX와 모바일 대응

- 모바일 세로 화면에서 한 화면 조작이 가능하도록 구성했습니다.
- 상단 HUD는 현재 주문과 지금 할 일만 보여줍니다.
- 하단 버튼은 다시, 도움, 소리 3개만 남겼습니다.
- 코치형 튜토리얼은 발사, 키우기, 배달을 직접 체험하게 합니다.
- 중앙 재료와 위쪽 배달칸은 강한 링과 문구로 강조됩니다.

## 6. 완성도와 활용성

- GitHub Pages로 즉시 배포할 수 있습니다.
- Matter.js를 로컬 파일로 포함해 별도 설치 없이 실행됩니다.
- 로컬 TOP 5, 오늘 챌린지, 결과 복사를 지원합니다.
- 캐릭터, 상점, 업적 요소로 반복 플레이 동기를 제공합니다.
- README, 기획서, 제출 ZIP을 함께 준비했습니다.

## 7. 왜 수상 후보인가

- 룰이 짧습니다: 쏘고, 중앙 재료를 맞춰 키우고, 위 칸에 배달합니다.
- 조작이 명확합니다: 누르고 조준하고 떼면 됩니다.
- 테마가 친근합니다: 도시락, 재료, 캐릭터가 즉시 이해됩니다.
- 모바일 대응이 되어 있습니다: 세로 화면 HUD와 3버튼 조작으로 단순합니다.
- 60초 안에 반복 욕구가 생깁니다: 성장, 배달, 콤보, 피버가 빠르게 순환합니다.

## PDF 제작 체크

- 첫 페이지 상단에 핵심 문장을 크게 배치합니다.
- 각 페이지는 한 메시지만 강조합니다.
- 스크린샷은 1페이지 또는 3페이지에 1장만 크게 넣습니다.
- 실제 제출 전 플레이 링크가 열리는지 확인합니다.
