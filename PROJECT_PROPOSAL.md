# 웹 미니게임 기획서: 빙글도시락 러시

## 1. 프로젝트 개요

- 게임명: 빙글도시락 러시
- 장르: 물리 기반 웹 미니게임 / 60초 타임어택
- 플랫폼: PC 및 모바일 브라우저
- 한 줄 소개: 회전하는 도시락 통 안에서 굴러다니는 재료를 알맞은 칸에 넣어 제한 시간 안에 주문 도시락을 완성하는 물리 액션 게임

빙글도시락 러시는 도시락 통을 좌우로 회전시키며 재료를 정해진 칸에 넣는 웹 미니게임입니다. 플레이어는 직접 재료를 잡거나 이동시키지 않고, 도시락 전체를 회전시켜 중력과 충돌을 이용해야 합니다. 단순한 조작으로 시작하지만, 재료가 굴러가고 튕기고 엉키는 물리 반응 때문에 매 판 다른 상황이 만들어지는 것이 핵심 재미입니다.

## 2. 기획 의도

웹 미니게임은 짧은 시간 안에 규칙이 이해되고, 바로 한 판 더 하고 싶어지는 구조가 중요하다고 판단했습니다. 그래서 익숙한 음식 소재인 도시락을 사용해 첫 화면만 봐도 목표를 이해할 수 있게 했고, 좌우 회전이라는 단순한 조작으로 물리 기반 손맛을 만들었습니다.

흔한 점프, 피하기, 벽돌깨기, 카드 매칭 방식 대신 “도시락 칸 정리”와 “회전 물리”를 결합해 차별화했습니다. 실패해도 재료가 우르르 굴러다니는 장면 자체가 재미있고, 성공하면 정확히 칸에 들어갔다는 즉각적인 만족감을 주는 게임을 목표로 합니다.

## 3. 핵심 컨셉

플레이어는 60초 동안 계속 들어오는 도시락 주문을 처리합니다. 각 주문에는 밥, 계란, 김치, 김, 새우 같은 재료와 필요한 개수가 표시됩니다. 도시락 내부에는 재료별 목표 칸이 있고, 플레이어는 도시락 통을 좌우로 회전시켜 재료가 해당 칸을 통과하게 해야 합니다.

맞는 칸에 재료가 들어가면 해당 재료가 바로 사라지고 주문 진행도가 올라갑니다. 반대로 다른 칸에 너무 오래 머물면 소폭 감점되고 콤보가 끊깁니다. 주문을 완성하면 새 주문이 즉시 등장하며, 제한 시간 동안 완성 수와 점수를 최대화하는 것이 목표입니다.

## 4. 주요 기능

- 실시간 물리 기반 재료 이동
  - 재료는 중력, 회전, 충돌에 따라 자연스럽게 굴러가고 튕깁니다.
  - 도시락 통을 과하게 회전하면 재료가 흩어져 리스크가 생깁니다.

- 회전 조작
  - PC에서는 `A/D` 또는 방향키로 조작합니다.
  - 모바일에서는 화면의 좌우 회전 버튼으로 조작합니다.
  - 조작은 단순하지만, 원하는 칸으로 재료를 보내려면 타이밍과 감각이 필요합니다.

- 주문 시스템
  - 매 주문마다 필요한 재료 구성이 달라집니다.
  - 주문을 완성할수록 다음 주문이 이어져 짧은 시간 안에 반복 플레이 흐름이 생깁니다.

- 점수 및 콤보 시스템
  - 맞는 칸에 재료를 넣으면 점수를 얻습니다.
  - 연속 성공 시 콤보가 올라가고 더 높은 점수를 받을 수 있습니다.
  - 잘못된 칸에 재료가 너무 오래 머물면 소폭 감점되고 콤보가 초기화됩니다.
  - 결과 화면에서는 기본 점수, 콤보 보너스, 주문 보너스, 아이템 보너스, 피버 보너스, 실수 감점이 분리되어 표시됩니다.

- 사운드 피드백
  - 성공, 실수, 콤보, 피버, 아이템, 게임 종료 순간에 서로 다른 효과음이 재생됩니다.
  - 효과음은 Web Audio API로 생성해 별도 사운드 파일 없이 동작합니다.

- 피버 타임
  - 콤보 x8 또는 도시락 3개 연속 완성 시 8초 동안 피버가 발동합니다.
  - 피버 중에는 점수가 2배가 되고 아이템이 더 자주 등장합니다.
  - 화면 색감과 파티클이 강화되어 보상 순간이 명확하게 느껴집니다.

- 오늘의 도시락 챌린지
  - 날짜를 기준으로 주문 순서와 아이템 배치를 고정합니다.
  - 서버 없이도 같은 날 같은 조건으로 점수 경쟁을 할 수 있습니다.
  - 일반 모드와 별도의 로컬 TOP 5를 제공합니다.

- 아이템 시스템
  - 플레이 중 반짝이는 보너스 아이템이 도시락 안에 등장합니다.
  - 재료가 아이템에 닿으면 시간 증가, 콤보 증가, 자석 보정 효과 중 하나가 발동합니다.
  - 아이템은 위험한 회전 상황에서도 역전 기회를 만들어 반복 플레이 동기를 강화합니다.

- 재료별 물성 차이
  - 밥은 무겁고 안정적으로 움직입니다.
  - 김치는 잘 튀고, 김은 가볍고 미끄럽게 움직입니다.
  - 계란은 잘 굴러가며, 새우는 반동이 크게 설정됩니다.

- 특수 주문
  - 빠른 주문은 10초 안에 완성하면 보너스 점수를 줍니다.
  - 깔끔 주문은 실수 없이 완성하면 보너스 점수를 줍니다.
  - 매운 주문은 김치 점수가 2배가 됩니다.
  - 단백질 주문은 계란과 새우만 등장합니다.
  - 무거운 주문은 밥 비중이 높고, 미끄러운 주문은 김 비중이 높습니다.
  - 금지칸 주문은 지정된 칸에 잘못된 재료가 오래 머물면 큰 감점이 발생합니다.

- 액티브 스킬
  - `Space` 또는 톡 치기 버튼으로 도시락 안 재료를 한 번 흔들 수 있습니다.
  - 쿨타임은 8초이며, 막힌 재료를 빼내는 긴급 수단으로 사용됩니다.

- 60초 결과 화면
  - 최종 점수, 완성한 도시락 수, 최고 콤보를 보여줍니다.
  - 점수 상세, 획득 코인, 새 업적, 오늘 챌린지 최고 기록 대비 차이를 보여줍니다.
  - 브라우저에 저장된 로컬 TOP 5 순위를 함께 보여줍니다.
  - 결과 공유 문구를 복사할 수 있는 버튼을 제공합니다.
  - 결과가 짧고 명확해 즉시 재도전하기 쉽습니다.

- 캐릭터와 상점
  - 플레이 결과로 코인을 획득하고, 코인으로 캐릭터와 소모 아이템을 구매합니다.
  - 캐릭터는 회전 속도, 점수 보정, 아이템 등장 간격, 자석 지속 시간, 시작 시간 등에 차이를 줍니다.
  - 캐릭터는 AI로 생성한 투명 PNG 일러스트를 사용해 상점에서 구분됩니다.
  - 선택한 캐릭터는 게임 화면에 표시되고 성공, 실수, 피버, 스킬 사용에 반응합니다.
  - 시작 전에는 캐릭터 선택 화면을 거쳐 원하는 캐릭터로 바로 플레이할 수 있습니다.
  - 상점 아이템은 진행 중인 판에서 시간 추가, 콤보 증가, 자석 효과, 즉시 피버 발동으로 사용할 수 있습니다.

- 미션과 업적
  - 피버 2회, 깔끔 주문 3회, 무실수 5연속, 아이템 5개, 최고 콤보 x12 같은 미션을 제공합니다.
  - 달성한 업적은 로컬 저장소에 기록되고 보상 코인을 지급합니다.
  - 별도 업적 컬렉션 화면에서 전체 업적 달성 상태와 보상을 확인할 수 있습니다.

- 밸런스 튜닝 모드
  - 캐릭터 성능 반영, 코인 지급량, 상점 가격, 주문 난이도, 아이템 등장 빈도, 특수 주문 빈도를 브라우저에서 조정할 수 있습니다.
  - 튜닝값은 로컬 저장소에 저장되며 기본값으로 복원할 수 있어 플레이 테스트와 난이도 조정이 쉽습니다.

- 사운드 설정
  - 효과음 ON/OFF 버튼을 제공하고 설정을 로컬 저장소에 유지합니다.

- 모바일 조작 보강
  - 모바일에서는 회전 버튼 외에도 게임 화면 좌우 절반을 터치해 회전할 수 있습니다.
  - 주요 성공, 실수, 아이템, 피버, 종료 순간에 기기 진동 피드백을 지원합니다.

## 5. 게임 흐름

1. 게임 시작과 동시에 첫 주문과 재료가 등장합니다.
2. 플레이어는 도시락 통을 좌우로 회전시켜 재료를 목표 칸으로 보냅니다.
3. 재료가 맞는 칸에 들어가면 바로 사라지고 주문 진행도가 올라갑니다.
4. 주문을 모두 채우면 점수 보너스를 받고 다음 주문으로 넘어갑니다.
5. 플레이 중 등장하는 아이템에 재료를 닿게 하면 보너스 효과를 얻습니다.
6. 콤보, 특수 주문, 피버, 톡 치기를 활용해 점수를 극대화합니다.
7. 60초가 끝나면 최종 결과와 로컬 순위가 표시되고 재시작하거나 결과를 복사할 수 있습니다.

## 6. 차별화 포인트

- 한 화면만 봐도 “도시락 재료를 칸에 넣는 게임”이라는 목표가 즉시 이해됩니다.
- 직접 이동이 아니라 통을 회전시키는 간접 조작이라 물리적인 손맛이 강합니다.
- 같은 주문이라도 재료 충돌과 회전 타이밍에 따라 매번 다른 상황이 발생합니다.
- 아이템, 피버, 특수 주문, 오늘 챌린지, 로컬 순위가 있어 단순 체험이 아니라 점수 경쟁과 재도전 흐름이 생깁니다.
- 귀여운 음식 소재와 타임어택 구조가 결합되어 짧은 심사 시간 안에 인상을 남기기 좋습니다.
- 설치나 빌드 없이 브라우저에서 바로 실행되는 정적 웹앱입니다.

## 7. 구현 범위

- HTML, CSS, JavaScript 기반 정적 웹앱으로 구현합니다.
- Matter.js 물리 엔진을 로컬 파일로 포함해 외부 설치 없이 실행합니다.
- PC와 모바일 브라우저 모두에서 플레이 가능하도록 반응형 UI를 제공합니다.
- 첫 버전에서는 브라우저 로컬 순위 경쟁에 집중하고, 서버 저장형 온라인 랭킹은 제외합니다.

## 8. 기대 효과

빙글도시락 러시는 누구나 이해할 수 있는 소재와 간단한 조작을 사용하지만, 실제 플레이에서는 물리 반응 때문에 예측하기 어려운 순간이 계속 발생합니다. 이로 인해 짧은 플레이 안에서도 웃긴 실패, 아슬아슬한 성공, 콤보 욕심이 자연스럽게 생깁니다.

해커톤 제출작으로는 “작지만 완성된 게임”을 목표로 하며, 심사자가 별도 설명 없이 바로 실행하고 재미를 확인할 수 있는 웹 미니게임을 지향합니다.
