# 빙글도시락 캐논

수박게임처럼 합치고, 캐논처럼 조준해서, 위쪽 주문칸에 배달하는 60초 도시락 러시입니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다. 물리 엔진은 `vendor/matter.min.js`에 포함되어 있어 별도 설치가 필요 없습니다.

로컬 서버로 확인하려면 프로젝트 폴더에서 아래 명령을 실행한 뒤 `http://127.0.0.1:4173/`을 엽니다.

```bash
python3 -m http.server 4173
```

## QA

Windows Chrome이 설치된 환경에서는 실제 브라우저 headless 모드로 데스크톱 키보드 조작 1판과 모바일 터치 조작 1판을 자동 확인할 수 있습니다.

```bash
node tools/browser-smoke.mjs http://127.0.0.1:4173/
```

## 게임 규칙

- 아래 캐논에서 재료를 조준 발사해 움직입니다.
- 발사대와 배달칸 사이에 잠깐 나타나는 같은 단계 타겟을 맞추면 현재탄이 한 단계 커집니다.
- `연습하고 시작`은 캐릭터 코치가 말풍선으로 설명하고, 화면에서 눌러야 할 대상만 밝게 보여주는 단계형 튜토리얼입니다.
- 첫 주문은 기본 배달과 합체를 짧게 배우도록 자동 도움을 줍니다.
- 모바일 상단 HUD는 현재 주문과 지금 할 일을 함께 표시합니다.
- 주문 음식이 위쪽 같은 재료 배달칸에 잠깐 머물면 배달되고 주문 진행도가 올라갑니다.
- 색 칸은 물리 벽이 아니라 배달 영역이라 재료가 도시락판 안에서 자연스럽게 흐릅니다.
- 첫 5개 주문을 완료한 뒤 나타나는 화살표 부스터에 닿으면 표시된 방향으로 재료가 튀어 판이 흔들립니다.
- 완성 단계가 되면 중앙 타겟이 사라지고, 위쪽 같은 재료 배달칸에 넣으면 배달됩니다.
- 빗나간 재료는 잠깐 뒤 자동 정리되어 바닥에 쌓이지 않습니다.
- 배달에 성공하면 도시락 도장 연출이 찍히고, 직배송/빙글배송 하이라이트가 기록됩니다.
- 첫 5개 주문 이후 반짝이는 아이템에 재료가 닿으면 시간, 콤보, 자석 효과를 얻습니다.
- 첫 판에서는 3개 배달 시 피버가 한 번 빠르게 등장하고, 이후에는 콤보 x8 또는 도시락 3개 연속 완성 시 피버 타임이 발동합니다.
- 특수 주문은 첫 8개 주문 이후 단계적으로 등장합니다.
- 다른 칸에 너무 오래 머물면 콤보가 끊깁니다.
- 60초 동안 완성한 도시락 수, 점수, 최고 콤보, 점수 상세와 로컬 TOP 5가 결과로 표시됩니다.

## 게임 가이드

첫 실행 시 `연습하고 시작`으로 코치형 연습모드에 들어갑니다. 캐릭터 말풍선을 읽고 화살표를 누르면 조작, 점수 구조, 자동발사 해금, 화살표 부스터, 난이도 변화를 차례로 설명합니다. 연습이 끝나면 60초 본 게임이 새로 시작됩니다. `바로 시작`을 누르면 기본 캐릭터로 즉시 시작하고, 첫 조작 전까지 타이머는 멈춥니다.
플레이 중에는 `가이드` 버튼 또는 `G` 키로 다시 볼 수 있습니다.
효과음은 Web Audio로 즉시 생성되며 별도 사운드 파일이 필요 없습니다. `사운드 ON/OFF` 설정은 로컬 저장소에 유지됩니다.

## 모드

- 일반 모드: 매 판 무작위 주문과 아이템 배치가 나옵니다.
- 오늘 챌린지: 날짜를 기준으로 주문 순서와 아이템 배치가 고정됩니다. 같은 날 같은 브라우저에서 TOP 5 경쟁을 할 수 있습니다.

## 아이템

- 시간 절임: 남은 시간을 5초 늘립니다.
- 콤보 소스: 현재 콤보를 3 올리고 보너스 점수를 줍니다.
- 자석 키트: 6초 동안 필요한 주문 음식 또는 합체 가능한 재료를 약하게 끌어당깁니다.

## 합체 단계

- 밥: 밥 → 주먹밥 → 김밥 → 특제 도시락
- 계란: 계란 → 계란말이 → 오믈렛 → 계란 도시락
- 김치: 김치 → 볶음김치 → 김치볶음밥 → 매운 도시락
- 김: 김 → 김말이 → 김 주먹밥 → 바삭 도시락
- 새우: 새우 → 새우튀김 → 새우볶음밥 → 해물 도시락

## 캐릭터와 상점

- 플레이 결과로 코인을 얻고, 코인으로 캐릭터와 소모 아이템을 구매합니다.
- 캐릭터는 발사 속도, 점수 보정, 아이템 등장 간격, 자석 지속 시간, 시작 시간 등에 차이를 줍니다.
- 캐릭터 일러스트는 `assets/characters/`의 투명 PNG로 상점에 표시됩니다.
- 선택한 캐릭터는 게임 화면에도 표시되며, 성공, 실수, 피버, 스킬 사용에 맞춰 말풍선 반응을 합니다.
- 상점 아이템은 진행 중인 판에서 시간 추가, 콤보 증가, 자석 효과, 즉시 피버 발동으로 사용할 수 있습니다.
- 구매, 보유 캐릭터, 인벤토리, 업적은 브라우저 로컬 저장소에 저장됩니다.

## 미션과 업적

- 피버 2회, 깔끔 주문 3회, 무실수 5연속, 아이템 5개, 최고 콤보 x12 같은 미션이 표시됩니다.
- 새 업적을 달성하면 결과 화면에 표시되고 보상 코인이 지급됩니다.
- `업적` 버튼에서 전체 업적 컬렉션과 달성 상태를 확인할 수 있습니다.

## 밸런스 튜닝

- 제출 화면에서는 튜닝 버튼을 숨깁니다. `?debug=1`로 실행하면 튜닝 버튼에서 캐릭터 성능 반영, 코인 지급량, 상점 가격, 주문 난이도, 아이템 등장 빈도, 특수 주문 빈도를 조정할 수 있습니다.
- 튜닝값은 로컬 저장소에 저장되며, `기본값` 버튼으로 되돌릴 수 있습니다.

## 피버와 스킬

- 피버 타임: 콤보 x8 또는 도시락 3개 연속 완성 시 8초 동안 점수 보너스와 2발 발사가 적용되고 아이템이 더 자주 등장합니다.
- 자동발사: 자동발사 버튼 또는 `E` 키로 현재탄 상태에 맞춰 중앙 타겟이나 배달칸으로 즉시 발사합니다.
- 자동발사는 4초 동안 풀파워로 연속 발사하며, 쿨타임 후 다시 사용할 수 있습니다.
- 자동발사는 본 게임에서 도시락 2개를 완성하면 해금됩니다.

## 재료 물성

- 밥: 무겁고 안정적입니다.
- 김치: 잘 튑니다.
- 김: 가볍고 미끄럽습니다.
- 계란: 둥글어서 잘 굴러갑니다.
- 새우: 반동이 큽니다.

## 순위

게임 종료 후 오늘의 하이라이트, 다음 목표, 브라우저에 저장된 로컬 TOP 5가 표시됩니다. 같은 기기와 브라우저에서 최고 기록 경쟁을 할 수 있습니다.
오늘 챌린지에서는 같은 날짜 최고 기록 대비 점수 차이가 결과 화면에 표시됩니다.
결과 화면의 `결과 복사` 버튼으로 점수 공유 문구를 복사할 수 있습니다.

## 배포

빌드 과정이 없는 정적 웹앱입니다. 저장소 루트를 그대로 배포 대상으로 지정하면 됩니다.

- GitHub Pages: 저장소 Settings → Pages에서 브랜치와 루트 폴더를 선택합니다. `.nojekyll`이 포함되어 있습니다.
- Netlify: 새 사이트 생성 후 publish directory를 `.`로 둡니다. `netlify.toml`이 포함되어 있습니다.
- Vercel: Framework Preset을 Other로 두고 Output Directory를 비워 둡니다.

## 제출 ZIP

제출용 압축파일은 아래 명령으로 만듭니다. `.git`, `.chrome-qa`, 로그 파일은 포함하지 않습니다.

```bash
tools/make-submit-zip.sh
```

## 조작

- 키보드: `Space`로 현재 조준 방향 발사, `A` / `D` 또는 `←` / `→`로 보조 자동 조준 발사
- 자동발사: `E` 또는 자동발사 버튼
- 터치/마우스: 캔버스를 누른 채 방향을 맞추고 게이지가 찬 순간 손을 떼어 발사
- 재시작: `R` 또는 재시작 버튼

## 제출 포인트

- 한 화면에서 룰이 바로 읽히는 도시락 캐논 합체 액션입니다.
- 흔한 점프/피하기/벽돌깨기 계열을 피하고, 수박게임식 합체와 주문 배달을 결합했습니다.
- 빌드 과정 없는 정적 웹앱이라 GitHub Pages, Netlify, Vercel 등에 바로 배포할 수 있습니다.
