# 웹 미니게임 기획서: 빙글도시락 러시

## 1. 프로젝트 개요

- 게임명: 빙글도시락 러시
- 장르: 물리 기반 합체 퍼즐 / 60초 타임어택
- 플랫폼: PC 및 모바일 브라우저
- 한 줄 소개: 젓가락 플리퍼로 재료를 튕기고, 같은 음식끼리 합쳐 주문 음식을 만든 뒤 도시락 칸에 배달하는 웹 미니게임

빙글도시락 러시는 굴러다니는 재료를 같은 단계끼리 합쳐 더 큰 음식으로 만들고, 현재 주문에 맞는 완성 음식을 같은 재료 칸에 배달하는 게임입니다. 조작은 좌우 젓가락 플리퍼와 톡 치기 스킬로 단순하지만, 어떤 재료를 먼저 합칠지, 큰 음식을 언제 배달할지 판단해야 해서 한 판 안에 작은 선택이 계속 생깁니다.

## 2. 기획 의도

수박게임, 뿌요뿌요, 테트리스처럼 단순한 규칙이 오래 재미있는 이유는 결과가 화면에 누적되고, 다음 선택이 이전 선택의 영향을 받기 때문입니다. 기존 도시락 물리 액션의 재료, 주문, 칸, 아이템 구조는 유지하되, 단순히 칸을 지나가면 점수가 오르는 구조를 버리고 합체와 배달을 핵심 재미로 바꿨습니다.

플레이어는 재료를 직접 잡지 않고 젓가락으로 튕깁니다. 같은 음식끼리 부딪히면 단계가 올라가고, 주문 음식이 같은 재료 칸에 잠깐 머물면 배달됩니다. 무작정 한 방향 입력을 유지해도 자동으로 점수가 오르지 않으며, 재료 합체와 공간 관리가 점수의 중심이 됩니다.

## 3. 핵심 루프

1. 기본 재료가 도시락 안에 투입됩니다.
2. 플레이어는 좌우 젓가락 플리퍼로 재료를 튕깁니다.
3. 같은 재료와 같은 단계 음식이 충돌하면 다음 단계 음식으로 합쳐집니다.
4. 현재 주문에 맞는 음식이 같은 재료 칸에 0.45초 머물면 배달 성공입니다.
5. 배달, 연속 합체, 피버, 아이템을 활용해 60초 동안 점수를 극대화합니다.

## 4. 주요 기능

- 재료별 합체 단계
  - 밥: 밥알, 주먹밥, 김밥, 특제 도시락
  - 계란: 계란, 계란말이, 오믈렛, 계란 도시락
  - 김치: 김치, 볶음김치, 김치볶음밥, 매운 도시락
  - 김: 김, 김말이, 김 주먹밥, 바삭 도시락
  - 새우: 새우, 새우튀김, 새우볶음밥, 해물 도시락

- 젓가락 플리퍼 조작
  - PC에서는 `A/D` 또는 방향키로 좌우 플리퍼를 튕깁니다.
  - 모바일에서는 화면 좌우 절반 또는 좌우 버튼으로 조작합니다.
  - 입력을 계속 누르고 있어도 반복 점수가 나지 않고, 순간 타이밍이 중요합니다.

- 주문 배달 시스템
  - 주문은 특정 재료와 합체 단계를 요구합니다.
  - 초반 주문은 기본 재료로 시작하고 이후 주문에서 합체 단계 요구가 올라갑니다.
  - 주문 음식이 같은 재료 칸에 안정적으로 머물러야 배달됩니다.
  - 재료 칸은 물리 칸막이가 아니라 배달 영역으로 처리해 재료가 플리퍼 쪽으로 흐릅니다.
  - 큰 음식일수록 점수가 높지만 공간을 더 차지합니다.

- 점수, 콤보, 피버
  - 합체와 배달 모두 점수를 줍니다.
  - 연속 합체와 배달로 콤보가 올라갑니다.
  - 콤보 x8 또는 주문 3개 연속 완성 시 피버가 발동하고 8초 동안 점수가 2배가 됩니다.

- 아이템과 스킬
  - 시간 절임은 남은 시간을 늘립니다.
  - 콤보 소스는 현재 콤보를 올립니다.
  - 자석 젓가락은 주문 음식 또는 합체 가능한 재료를 끌어당깁니다.
  - 톡 치기 스킬은 막힌 재료를 한 번 흔드는 긴급 수단입니다.

- 캐릭터와 상점
  - 캐릭터는 플리퍼 힘, 점수 보정, 아이템 등장 간격, 자석 지속 시간, 시작 시간 등에 차이를 줍니다.
  - 결과 보상 코인으로 캐릭터와 소모 아이템을 구매합니다.
  - 보유 캐릭터, 인벤토리, 업적은 브라우저 로컬 저장소에 유지됩니다.

- 모드와 순위
  - 일반 모드는 매 판 무작위 주문과 아이템 배치가 나옵니다.
  - 오늘 챌린지는 날짜 기준으로 주문과 아이템 난수를 고정합니다.
  - 결과 화면에는 점수 상세, 획득 코인, 업적, 로컬 TOP 5, 결과 복사 문구가 표시됩니다.

## 5. 차별화 포인트

- 수박게임식 합체의 직관성과 물리 기반 도시락 액션을 결합했습니다.
- 재료가 사라지기만 하는 게임이 아니라, 화면 안에 음식이 누적되어 다음 판단을 만듭니다.
- 단순 조작이지만 합체 순서, 배달 타이밍, 공간 관리가 점수를 좌우합니다.
- 설치나 빌드 없이 브라우저에서 바로 실행되는 정적 웹앱입니다.

## 6. 구현 범위

- HTML, CSS, JavaScript 기반 정적 웹앱입니다.
- Matter.js 물리 엔진을 로컬 파일로 포함합니다.
- PC와 모바일 브라우저 모두에서 플레이할 수 있습니다.
- 서버 저장형 온라인 랭킹은 제외하고 브라우저 로컬 기록에 집중합니다.

## 7. 기대 효과

빙글도시락 러시는 음식 소재의 직관성과 합체 퍼즐의 반복 욕구를 함께 노립니다. 플레이어는 “조금만 더 합치면 큰 음식이 된다”는 욕심과 “지금 배달해야 점수를 얻는다”는 타이밍 판단 사이에서 계속 선택하게 됩니다. 짧은 플레이 안에서도 합체, 실수, 피버, 주문 배달이 빠르게 반복되어 제출용 미니게임으로 완성감을 줄 수 있습니다.
