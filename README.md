# 빙글도시락 러시

60초 안에 주문 도시락을 최대한 많이 완성하는 웹 미니게임입니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다. 물리 엔진은 `vendor/matter.min.js`에 포함되어 있어 별도 설치가 필요 없습니다.

로컬 서버로 확인하려면 프로젝트 폴더에서 아래 명령을 실행한 뒤 `http://127.0.0.1:4173/`을 엽니다.

```bash
python3 -m http.server 4173
```

## QA

Windows Chrome이 설치된 환경에서는 실제 브라우저 headless 모드로 데스크톱 키보드 조작 1판과 모바일 터치 조작 1판을 자동 확인할 수 있습니다.

```bash
node tools/browser-smoke.mjs http://127.0.0.1:4173/
```

## 게임 규칙

- 도시락 통을 좌우로 회전해 재료를 같은 이름의 칸에 넣습니다.
- 재료가 맞는 칸에 들어가면 바로 사라지고 주문 진행도가 올라갑니다.
- 반짝이는 아이템에 재료가 닿으면 시간, 콤보, 자석 효과를 얻습니다.
- 콤보 x8 또는 도시락 3개 연속 완성 시 피버 타임이 발동해 8초 동안 점수가 2배가 됩니다.
- 특수 주문은 빠른 주문, 깔끔 주문, 매운 주문, 단백질 주문, 무거운 주문, 미끄러운 주문, 금지칸 주문으로 등장합니다.
- 다른 칸에 너무 오래 머물면 소폭 감점되고 콤보가 끊깁니다.
- 60초 동안 완성한 도시락 수, 점수, 최고 콤보, 점수 상세와 로컬 TOP 5가 결과로 표시됩니다.

## 게임 가이드

첫 실행 시 가이드가 먼저 표시됩니다. 가이드에서 시작하면 캐릭터 선택 화면을 거친 뒤 타이머가 시작됩니다.
플레이 중에는 `가이드` 버튼 또는 `G` 키로 다시 볼 수 있습니다.
효과음은 Web Audio로 즉시 생성되며 별도 사운드 파일이 필요 없습니다. `사운드 ON/OFF` 설정은 로컬 저장소에 유지됩니다.

## 모드

- 일반 모드: 매 판 무작위 주문과 아이템 배치가 나옵니다.
- 오늘 챌린지: 날짜를 기준으로 주문 순서와 아이템 배치가 고정됩니다. 같은 날 같은 브라우저에서 TOP 5 경쟁을 할 수 있습니다.

## 아이템

- 시간 절임: 남은 시간을 5초 늘립니다.
- 콤보 소스: 현재 콤보를 3 올리고 보너스 점수를 줍니다.
- 자석 젓가락: 6초 동안 재료가 맞는 칸 쪽으로 약하게 끌립니다.

## 캐릭터와 상점

- 플레이 결과로 코인을 얻고, 코인으로 캐릭터와 소모 아이템을 구매합니다.
- 캐릭터는 회전 속도, 점수 보정, 아이템 등장 간격, 자석 지속 시간, 시작 시간 등에 차이를 줍니다.
- 캐릭터 일러스트는 `assets/characters/`의 투명 PNG로 상점에 표시됩니다.
- 선택한 캐릭터는 게임 화면에도 표시되며, 성공, 실수, 피버, 스킬 사용에 맞춰 말풍선 반응을 합니다.
- 상점 아이템은 진행 중인 판에서 시간 추가, 콤보 증가, 자석 효과, 즉시 피버 발동으로 사용할 수 있습니다.
- 구매, 보유 캐릭터, 인벤토리, 업적은 브라우저 로컬 저장소에 저장됩니다.

## 미션과 업적

- 피버 2회, 깔끔 주문 3회, 무실수 5연속, 아이템 5개, 최고 콤보 x12 같은 미션이 표시됩니다.
- 새 업적을 달성하면 결과 화면에 표시되고 보상 코인이 지급됩니다.
- `업적` 버튼에서 전체 업적 컬렉션과 달성 상태를 확인할 수 있습니다.

## 밸런스 튜닝

- `튜닝` 버튼에서 캐릭터 성능 반영, 코인 지급량, 상점 가격, 주문 난이도, 아이템 등장 빈도, 특수 주문 빈도를 조정할 수 있습니다.
- 튜닝값은 로컬 저장소에 저장되며, `기본값` 버튼으로 되돌릴 수 있습니다.

## 피버와 스킬

- 피버 타임: 콤보 x8 또는 도시락 3개 연속 완성 시 8초 동안 점수가 2배가 되고 아이템이 더 자주 등장합니다.
- 톡 치기: `Space` 또는 톡 치기 버튼으로 재료를 한 번 흔듭니다. 쿨타임은 8초입니다.

## 재료 물성

- 밥: 무겁고 안정적입니다.
- 김치: 잘 튑니다.
- 김: 가볍고 미끄럽습니다.
- 계란: 둥글어서 잘 굴러갑니다.
- 새우: 반동이 큽니다.

## 순위

게임 종료 후 브라우저에 저장된 로컬 TOP 5가 표시됩니다. 같은 기기와 브라우저에서 최고 기록 경쟁을 할 수 있습니다.
오늘 챌린지에서는 같은 날짜 최고 기록 대비 점수 차이가 결과 화면에 표시됩니다.
결과 화면의 `결과 복사` 버튼으로 점수 공유 문구를 복사할 수 있습니다.

## 배포

빌드 과정이 없는 정적 웹앱입니다. 저장소 루트를 그대로 배포 대상으로 지정하면 됩니다.

- GitHub Pages: 저장소 Settings → Pages에서 브랜치와 루트 폴더를 선택합니다. `.nojekyll`이 포함되어 있습니다.
- Netlify: 새 사이트 생성 후 publish directory를 `.`로 둡니다. `netlify.toml`이 포함되어 있습니다.
- Vercel: Framework Preset을 Other로 두고 Output Directory를 비워 둡니다.

## 조작

- 키보드: `A` / `D`, `←` / `→`
- 스킬: `Space`
- 터치: 화면 오른쪽의 회전 버튼
- 재시작: `R` 또는 재시작 버튼

## 제출 포인트

- 한 화면에서 룰이 바로 읽히는 도시락 물리 액션입니다.
- 흔한 점프/피하기/벽돌깨기 계열을 피하고, 회전하는 칸과 굴러다니는 재료의 조합으로 차별화했습니다.
- 빌드 과정 없는 정적 웹앱이라 GitHub Pages, Netlify, Vercel 등에 바로 배포할 수 있습니다.
